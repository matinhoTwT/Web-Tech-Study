# Web-Tech-Study

Projeto com exemplos de HTML, CSS e JavaScript para estudo e prática.

## Sobre

Este repositório contém exercícios simples de front-end e exemplos de código JavaScript.

## Estrutura do projeto

- `Exemplo Js/`
  - `calc.js` - operações matemáticas básicas em JavaScript.
  - `exer1.js` - exercício JavaScript.
  - `exer2.js` - exercício JavaScript.
  - `primeiro.js` - exemplo inicial de JavaScript.
  - `textp.js` - exemplo de manipulação de texto.
  - `tipos-dados.js` - exemplos de tipos de dados.
- `Form Web/`
  - `index.html` - página de formulário.
  - `css/` - estilos do site.
  - `JavaScript/` - scripts para o formulário.

## Como usar

1. Abra o projeto em um editor de código (por exemplo, VS Code).
2. Abra os arquivos HTML no navegador para ver a interface.
3. Execute os arquivos JavaScript com `node` ou use o console do navegador para testar.

## Tecnologias

- HTML
- CSS
- JavaScript
